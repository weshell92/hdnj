# 黄帝内经 · 生命经典知识系统

一个围绕《黄帝内经》构建的中医经典知识系统网站。

## 项目目标

- 原典阅读：完整整理《素问》81篇、《灵枢》81篇目录，并支持从 CText 原典源动态读取全文。
- 知识体系：阴阳五行、天人相应、脏腑经络、气血津液、病因病机、治则治法。
- 经络图谱：完整十二经络循行图谱、流注时间、主要穴位、病候关键词与原文依据。
- 大众入口：四季养生、扩充版名句精读、学习路径、古今对照。
- 安全边界：AI 读书助手只做经典学习，不做诊断、开方或治疗建议。

## 本地运行

## 原典数据说明

项目内置《素问》《灵枢》共 162 篇完整目录、关键词、导读和核心种子原文。篇章详情页会优先通过 `src/lib/ctext.ts` 从 CText 原典源读取完整原文；如果部署环境无法联网，则展示本地种子文本，并提供“查看 CText 原典来源”外链。

这样做可以避免手工复制全书造成遗漏，同时保留稳定目录、搜索和学习体验。

## 医学边界

本站内容用于传统文化学习、经典阅读与中医思想理解，不构成医疗诊断、治疗或用药建议。如有身体不适，请咨询正规医疗机构或专业医生。
