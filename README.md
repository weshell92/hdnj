# 黄帝内经 · 生命经典知识系统

一个围绕《黄帝内经》构建的中医经典知识系统网站。

## 项目目标

- 原典阅读：《素问》《灵枢》篇章阅读、白话、关键词、思想解读。
- 知识体系：阴阳五行、天人相应、脏腑经络、气血津液、病因病机、治则治法。
- 大众入口：四季养生、名句精读、学习路径、古今对照。
- 安全边界：AI 读书助手只做经典学习，不做诊断、开方或治疗建议。

## 本地运行

```bash
npm install
npm run dev -- -p 3003
```

访问：

```text
http://localhost:3000
```

## 构建

```bash
npm run build
npm run start
```

## 主要功能

- 首页生命体系图
- 《素问》《灵枢》阅读
- 知识体系地图
- 阴阳五行专题
- 五脏六腑专题
- 十二经络专题
- 四季养生专题
- 名句精读
- 学习路径
- 古今对照
- 全站搜索
- AI 读书助手安全占位版
- 医学免责声明

## 医学边界

本站内容用于传统文化学习、经典阅读与中医思想理解，不构成医疗诊断、治疗或用药建议。如有身体不适，请咨询正规医疗机构或专业医生。
